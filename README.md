# TEC - Inteligencia Artificial Empresarial

Repositorio del curso **Inteligencia Artificial Empresarial** del Tecnológico de Costa Rica (TEC). Aquí se almacenan los materiales, datasets y notebooks desarrollados en **Google Colab**, como parte de los trabajos autónomos y colaborativos del curso.

---

## 🎯 Objetivo

Aplicar técnicas de Inteligencia Artificial en problemas empresariales reales y simulados, con un enfoque práctico desde Google Colab. Se incluyen análisis exploratorio, modelos predictivos, visualización de datos, clasificación y regresión usando bibliotecas de Python.

---

## 🗂️ Estructura del repositorio

```bash
TEC-IA-Empresarial/
├── datasets/              # Archivos CSV o datos usados en ejercicios
├── notebooks/             # Notebooks de Google Colab (.ipynb)
├── docs/                  # Documentación complementaria (opcional)
└── README.md              # Descripción general del repositorio
```

---

## 🚀 ¿Cómo usar este repositorio?

### 📌 Opción 1: Abrir directamente en Google Colab

Haga clic en el enlace del notebook deseado para abrirlo directamente en Colab.

```markdown
[![Abrir en Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/BarnaldBocker/TEC-IA-Empresarial/blob/main/notebooks/mi_notebook.ipynb)
```

> Reemplace `mi_notebook.ipynb` con el nombre de su notebook.

---

### 📌 Opción 2: Ejecutar en su entorno local (opcional)

Si desea trabajar localmente:

```bash
git clone https://github.com/BarnaldBocker/TEC-IA-Empresarial.git
cd TEC-IA-Empresarial
```

Instale los paquetes necesarios:

```bash
pip install -r requirements.txt
```
